Acerca del Proyecto

Este es un proyecto de titulación que presenta un prototipo funcional de un sistema de gestión de inventario para una clínica veterinaria. El objetivo principal es demostrar las capacidades de desarrollo web moderno y servir como base sólida para futuras implementaciones y mejoras.

⚠️ Aclaración Importante: Este sistema es un demo/prototipo y no está implementado en producción. Está diseñado como una prueba de concepto y base arquitectónica para futuros desarrollos reales.

🎯 Objetivo

Demostrar conocimientos en desarrollo web full-stack

🔧 Base Técnica

Estructura lista para expansión y escalabilidad

📈 Futuro

Plantilla para implementación real futura

Características del Sistema

📦 Gestión de Productos

Administración completa de productos con categorías, precios, códigos de identificación y estados de disponibilidad.

📊 Control de Stock

Monitoreo en tiempo real de existencias, alertas de stock bajo y próximos a vencer, y registro de movimientos.

👥 Gestión de Proveedores

Base de datos de proveedores con información de contacto e historial de suministros.

📋 Categorías

Clasificación organizada de productos por tipo (medicinas, equipos, suministros, etc.).

🔐 Panel Administrativo

Interfaz segura para gestionar usuarios, permisos y configuraciones del sistema.

⚙️ API REST

Interfaz programática para integración con otros sistemas futuros.
